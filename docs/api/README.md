# 平板点餐系统后台接口文档

#### 统一请求返回内容（JSON）

无论请求是否处理成功均返回内容，内容模板如下：

```json
{
  "success": true,      // 是否处理成功
  "message": "xxx",     // 错误信息
  "code": 20000,        // 响应码
  "payload": {          // 主要内容
    ...
  }
}
```

#### 响应码

| 数值  | 含义     |
| ----- | -------- |
| 20000 | 请求成功 |
| 40000 | 请求失败 |
